Cursos Preferidos

Ingeniería de Software I

Este curso es muy importante, pues aprendí no sólo a manejar herramientas de desarrollo de software, también aprendí sobre metodologías ágiles y cómo estas se aplican a los proyectos de desarrollo de software.

Ingeniería de Software II

Este curso, que es la continuación de Ingeniería de Software I, se profundiza más en la cercanía con el mundo real ya que tuvimos un cliente real, lo que me permitió darme una idea más clara del mercado laboral.

Bases de Datos I

En este curso aprendí a utilizar herramientas necesarias para la gestión de bases de datos.

Bases de Datos II

Este curso es una continuación de bases de datos I en el cual logré explorar más a profundidad herramientas y conceptos relacionados con las bases de datos.

Desarrollo de Aplicaciones de Internet

En este curso se aprende las herramientas y conocimientos básicos de HTML y CSS (y frameworks de interfaz de usuario como bootstrap y Foundation) que se debe saber para el diseño y desarrollo de páginas web.

Graficación Interactiva I

En este curso se aprenden conceptos básicos de graficación interactiva (software de manejo de gráficos, hardware como tarjetas gráficas). Me gusta mucho este curso ya que puedo explorar mi lado creativo y ver la computación como un medio para transmitir arte. Además que va muy de la mano con la creación de videojuegos.
